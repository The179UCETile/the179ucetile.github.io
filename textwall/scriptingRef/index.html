Window properties

unreadAmount

Returns the amount of unread messages (BROKEN)

cel()

Returns the cursor's internal decoration ID.

undoWrite()

Removes the most recent character(s) written in the canvas.

redoWrite()

Adds back the most recent character(s) written in the canvas.

selectedChatTab

Returns the selected chat channel.
Return values:

  • 0: This wall
  • 1: Global

accept_nsfw()

Hides the NSFW message By using this command, you understand that the wall you're about to enter contains NSFW content.

decline_nsfw()

Declines the NSFW message and go back to the main wall

printMsg(nick, color, msg, isRegistered, isAdmin, channel, displayNick, timestamp, tag, html = false)

Displays a message only shown to the client.
Parameters:

  • nick: The user who sent the message.
  • color: The username color, using internal syntax.
  • msg: The message to send.
  • isRegistered: Determines if the user is registered or not.
  • isAdmin: Doesn't do anything.
  • channel: The channel to send in.
  • displayNick: Overwrites nick.
  • timestamp: The time (given by Date.now()) when the message is sent.
  • tag: The tag. You can use HTML to style the tag.
  • html: Doesn't do anything.
