TextWall scripting reference

NOTE: This is a work in progress!

Window porperties

unreadAmount

Returns the amount of unread messages (BROKEN)

cel()

Returns the cursor's internal decoration ID.

undoWrite()

Removes the most recent character(s) written in the canvas.

redoWrite()

Adds back the most recent character(s) written in the canvas.

selectedChatTab

Returns the selected chat channel.
Return values:

  • 0: This wall
  • 1: Global
